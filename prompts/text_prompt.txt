다음 텍스트에서 학습 주제를 여러 개로 나누어라.

반드시 JSON 형식으로 반환하라.

형식:

{
  "topics": [
    {
      "topic": "",
      "content": "",
      "week": 1
    }
  ]
}

규칙:

- topic은 핵심 주제 이름이다.
  예: 이진탐색, 정렬, DFS, BFS, 재귀 등

- topic은 여러 개여도 된다.

- content는 해당 topic의 핵심 내용을 충분히 포함해야 한다.

- week는 학습 순서를 기준으로 추정한다.
  예: 첫 번째 개념이면 1

중요:

- 반드시 topics 배열을 포함해야 한다.
- JSON 외 다른 텍스트는 절대 출력하지 마라.
- 주제가 여러 개면 여러 객체로 나누어라.
